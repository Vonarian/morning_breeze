# 🌬️ Morning Breeze

> "Why are you gay?" — Simon Kaggwa Njala

A shell-based reenactment of the legendary 2012 interview between Simon Kaggwa Njala and Pepe Julian Onziema. This script brings the dramatic pauses, the intense staring, and the "trans-gender" confusion directly to your terminal.

## 🚀 Features

- **Dynamic Interaction:** You play as Pepe; you must defend your identity.
- **Dramatic Visuals:** ANSI color coding and ASCII "soul-staring" eyes.
- **Director's Commentary:** Subtle hints to keep the scene moving.
- **Surprise Guest:** Features the sudden entry of Pastor Martin Ssempa.

## 🛠️ Usage

This repository contains multiple variants of the same interactive experience:

### Bash

```bash
./morning_breeze.sh
```

_Note: Make sure the script is executable (`chmod +x morning_breeze.sh`)._

### Python

```bash
python3 morning_breeze.py
```

### Go

```bash
go run morning_breeze.go
```

### Dart

```bash
dart run morning_breeze.dart
```

### Node.js

```bash
node morning_breeze.js
```

### Windows Batch

Simply run `morning_breeze.bat` in your Windows command prompt (requires Windows 10+ for color support).
